FinTrak is a finance budgeting/tracker tool created to monitor your spending and income at the simplest level

A working skeleton at the moment, mainly created to explore some web dev work. Extra features can be added in the future

Done on Python 3.11,needs to be ran locally with the frameworks, needs a PostgeSQL db, scaffold of db is provided but needs to have the same name in the engine, all the password enrypticon and passwords are included out of .gitignore as webapp this isnt hosted anywhere
